# Videre
OpenGL examples written in julia.

# Usage
Please set up OpenGL version before running these examples.

| Platform | Default GL Version |
| :--------|:------------------:|
| OSX                     | 4.1 |
| Linux                   | 4.4 |
| Windows                 | 4.4 |
